# Recipe Finder

A recipe search application built with TypeScript, HTML, and CSS.

## Features (Coming Soon!)
- Search for recipes
- View recipe details
- Filter by diet and cuisine
- Save favorites

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown in the terminal (usually http://localhost:5173)

## Build for Production

```bash
npm run build
```

## Technologies
- TypeScript
- HTML5
- CSS3
- Vite (build tool)
- TheMealDB API
